Задание:
Создайте систему управления контактами:
1. Список контактов:
• Показывает контакты пользователя.
• Позволяет добавлять, редактировать и удалять контакты.
2. Профиль пользователя:
• Показывает информацию о пользователе.
• Позволяет редактировать профиль.
Требования:
• Backend (Laravel):
• Создайте API для управления контактами и профилем пользователя.
• Упакуйте backend в контейнер Docker.
• Frontend (Vue.js):
• Создайте интерфейс для списка контактов и профиля пользователя.
• Упакуйте frontend в контейнер Docker.
• Дополнительно:
• Реализуйте аутентификацию пользователя.
• Добавьте возможность поиска и фильтрации контактов.
• Упакуйте весь проект (frontend, backend, базу данных) в единый Dockercompose файл.
• Git:
• Залейте исходный код на Git репозиторий.
